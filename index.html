CODESQAUD COMICS

CodeSqaud Comics is a collection of graphic novels read by Quentin Brandel. This site is intended to display comic book covers along with information about each book, including the author, a rating, and other details about the graphic novel. Browse through the complete collection below. Click on the cover image or the Details link to see even more information for each graphic novel including the publisher, genre, number of pages, and a brief synopsis. The About page includeds meta information about this collection. Login is only available to the site adminstrator at this time.
